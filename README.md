# Photonic Integrated Circuits — Typed Lecture Notes

Typed-up notes transcribed from the scanned handwritten lecture notes for **EECE 7398 — Analysis & Design of Photonic Integrated Circuits (PICs)** (Northeastern University, ECE).

Each lecture is a standalone Markdown file with LaTeX math and the original hand-drawn figures cropped from the scans and embedded. Figures live in `figures/<lecture-number>/`.

> Tip: view these in a Markdown viewer with math support (e.g. VS Code with a Markdown+Math extension, Obsidian, or Typora) so the `$...$` / `$$...$$` equations render.
